## 資料夾架構
 - public # 不需編譯靜態檔
 - src # 主原始碼
    - assets # 需編譯靜態檔
    - components # 元件放置處
    - layouts # 基礎頁面呈現配置放置處
    - locales # i18n
    - plugins # 三方外掛
    - middleware # 中介
    - store # pinia
    - pages # 頁面
    
## 指令

### pnpm
- 安裝步驟
```
pnpm i 
```

- 啟動本地端
```
pnpm run dev 
```

- 打包專案
```
pnpm build 
```

## 技術棧
-	css框架：twailwind
-	狀態管理：Pinia
-   多語系: nuxt@i18n
-	後端服務：laravel
-	建構工具：Vite
